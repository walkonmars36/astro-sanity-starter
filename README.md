# Astro + Sanity Starter

A clean scaffold for Astro (SSR) + Sanity with Visual Editing pre-configured. Built by 13foot3 as a repeatable starting point for client projects.

Includes: SSR with Node adapter, @sanity/astro integration, draft mode cookie plumbing, SanityVisualEditing and DisableDraftMode components, shared layout, and a hello world page proving the pipeline end to end.

Pair with: sanity-studio-starter
